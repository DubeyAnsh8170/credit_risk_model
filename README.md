# Credit Risk Modelling & Scorecard Development

## Project Overview
This project builds a Probability of Default (PD) model to assess borrower credit risk using historical loan data.

## Objective
To predict the likelihood of loan default and design a credit scorecard for risk-based decision making.

## Techniques Used
- Logistic Regression
- Weight of Evidence (WOE)
- Information Value (IV)
- Credit Scorecard Development
- ROC-AUC
- KS Statistic

## Tools
Python, Pandas, NumPy, Scikit-learn, Matplotlib

## Model Workflow
1. Data Cleaning
2. Feature Engineering
3. WOE Binning
4. IV Calculation
5. Logistic Regression Model
6. Model Evaluation (ROC, KS)
7. Scorecard Development

## Results
The model successfully classifies borrowers into risk categories and demonstrates strong discriminatory power using ROC-AUC and KS metrics.
